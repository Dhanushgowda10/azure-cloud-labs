# Azure Cloud Labs

A comprehensive collection of hands-on Azure cloud deployment guides designed for IT professionals, DevOps engineers, and cloud practitioners.

## Overview

This repository contains step-by-step lab guides for deploying and managing Azure cloud resources. Each guide includes detailed instructions, best practices, and configuration settings for real-world cloud deployments.

## Lab Guides

### 1. Azure App Service Deployment
**[app-service-deployment-guide.md](app-service-deployment-guide.md)**

Learn how to deploy a Java-based web application using Azure App Service with continuous deployment from GitHub.

**Topics Covered:**
- Creating and configuring Azure App Service
- Setting up Java runtime environment
- Integrating GitHub for CI/CD
- Managing web application settings

### 2. Azure Virtual Machine Deployment
**[vm-deployment-guide.md](vm-deployment-guide.md)**

Deploy and configure Windows Server Virtual Machines in Azure with proper networking and security settings.

**Topics Covered:**
- Creating VMs in Azure portal
- Configuring compute, storage, and networking
- Setting up RDP access
- Best practices for VM management

## Technologies Used

- Microsoft Azure
- Azure App Service
- Azure Virtual Machines
- GitHub Actions (CI/CD)
- Windows Server
- Java Runtime

## Prerequisites

- Azure subscription (free tier works)
- Basic understanding of cloud concepts
- GitHub account (for CI/CD integration)
